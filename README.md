# FFXIKeys
A small windower addon to automate trading Special Gobbiedial Keys for rewards

Load Addon
  * //lua l ffxikeys
  
Unload Addon
  * //lua u ffxikeys
  
Use Keys
  * //keys unlock <key> <target>
  
Buy Keys
  * //keys buy <key> <target> <count>
  
Stop
  * //keys stop
  
Toggle Printing Links
  * //keys printlinks

Toggle Opening Links
  * //keys openlinks
